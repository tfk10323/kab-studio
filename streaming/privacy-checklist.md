# Privacy Checklist for Streaming

## ALWAYS HIDE (Use Privacy Scene)

### Environment & Secrets
- [ ] `.env` files
- [ ] API keys in any form
- [ ] Supabase dashboard (shows anon key)
- [ ] Square dashboard
- [ ] Any config files with secrets

### Customer Data
- [ ] Customer names
- [ ] Phone numbers
- [ ] Email addresses
- [ ] Delivery addresses
- [ ] Payment information

### Personal Accounts
- [ ] Email inbox
- [ ] Banking/financial sites
- [ ] Password managers

## SAFE TO SHOW
- [x] Code editors with public code
- [x] Public GitHub repos
- [x] Design files
- [x] Game development

## OBS HOTKEYS

| Key | Action |
|-----|--------|
| **F9** | Toggle Privacy Mode |
| **F10** | BRB Scene |
| **F11** | Main Coding Scene |
| **F12** | Mute/Unmute Mic |

## Pre-Stream Checklist
1. [ ] Close all sensitive browser tabs
2. [ ] Close email client
3. [ ] Close Supabase dashboard
4. [ ] Test privacy scene hotkey
