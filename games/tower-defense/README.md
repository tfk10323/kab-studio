# Tower Defense - KAB Games

**Status:** Design Phase

## Concept
4-player co-op tower defense inspired by CoD Zombies.

## Tech Stack
- **Engine:** Godot 4
- **Language:** GDScript
- **Players:** 3-4 (local + online co-op)

## Planned Features
- Wave-based enemy spawns
- Multiple tower types
- Player roles/classes
- Boss waves

## Asset Sources
- Kenney.nl (CC0)
- itch.io asset packs
- OpenGameArt.org
