# KAB Studio Color Palette

## Primary Colors

| Name | Hex | Usage |
|------|-----|-------|
| **KAB Gold** | `#ffc805` | Primary accent, CTAs |
| **KAB Purple** | `#8b5cf6` | Secondary accent, gradients |

## Background Colors

| Name | Hex | Usage |
|------|-----|-------|
| **Dark BG** | `#0a0a0f` | Primary background |
| **Dark Surface** | `#12121a` | Cards, modals |

## CSS Variables

```css
:root {
  --kab-gold: #ffc805;
  --kab-purple: #8b5cf6;
  --bg-primary: #0a0a0f;
  --bg-surface: #12121a;
}
